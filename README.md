# GraphQL API with Express & TypeScript

A modular GraphQL API built with **Express**, **TypeScript**, and **Prisma**, featuring Jest unit tests and a clean module-based architecture.

---

## Technologies

- Node.js & Express
- TypeScript
- GraphQL
- Prisma ORM
- PostgreSQL
- Jest for testing

---

## Features

- GraphQL Queries and Mutations
- Module-based architecture (users, posts, comments)
- Type-safe services with Prisma
- Unit tests with Jest
- Authentication support via context

---

## Getting Started

1. Clone the repository

```bash
git clone
```

2. Setup .env variables as in .env.example

3. For local development run

```bash
npm run dev
```

4. For running tests suits run

```bash
npm run test
```
